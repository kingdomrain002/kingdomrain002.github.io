启动Apache服务

Mac OS X自带了Apache，我们只需要简单的启动它就行了。使用如下命令:
启动：sudo apachectl start
停止：sudo apachectl stop
重启：sudo apachectl restart
查看 Apache 版本 httpd -v

开启PHP模块

Mac OS X已经内置PHP，因此我们只需要在Apache的配置中加载PHP模块即可。
打开Apache配置文件，路径如下:/etc/apache2/httpd.conf。
打开注释 LoadModule php7_module libexec/apache2/libphp7.so
然后，重启Apache服务。
