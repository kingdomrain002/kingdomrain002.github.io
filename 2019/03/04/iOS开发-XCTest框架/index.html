iOS开发-XCTest框架

Posted on 2019-03-04

前言

随着公司项目增多，自动化测试的需求也应运而生。本周我主要是学习如何基于苹果提供的测试框架是XCTest框架进行自动化测试。后面还要开发SDK，驱动XCTest来做自动化测试。

搜索引擎找到了很多XCTest框架使用的文章，但是仔细看下文章的内容却觉得太浅显。
参考资料《About Testing with Xcode》和《Framework XCTest》是苹果提供的文档。如果想系统学习XCTest框架，还是得仔细阅读苹果提供的文档。
苹果还提供了WWDC的视频,其中: WWDC 2015 UI Testing in Xcode 7 (406)就非常值得一看。

当然也有优秀的社区文档，例如objc期刊的这篇《XCTest 测试实战》，就是基于XCTest的工程实践。
